AddyCaddy

Moving to Denver or within Denver can be a game of speed, and waiting to sign a lease means you might lose out. AddyCaddy is designed to give prospective renters the ability to quickly compare two rental addresses based on criteria that is important to them.

    Ruby on Rails | PostgreSQL | OAuth | Open Data | Geocoding | APIs

  • ETL for data stored on data.denvergov.org
  • exposed versioned, documented API
  • Redis server and Resque gem manage email as a background job
  • Twitter OAuth to conduct user login

Habit Rabbit

Staying true to the things that are important to you takes more than sheer willpower; it requires habit. And habits require help. Habit Rabbit is a platform that allows you to track habits, receive optional notifications to encourage habit completion, and graphs showing your progress.

    Ruby on Rails | OAuth | Twilio | Cron Jobs | PostgreSQL | Foundation

  • optimized performance with cron jobs and background workers
  • Data-visualizations using Rickshaw
  • test driven feature implementation using Factory Girl, RSpec, Capybara

Airlift

Airlift pivots a legacy code base into a multi-tenant application similar to Etsy. To provide relief to customers in areas affected by disasters, Airlift provides customers with goods delivery via helicopter. Customers simply choose a location on a Google map where their supplies will be dropped.

    Ruby on Rails | Google Maps API | jQuery | Bootstrap 3

  • geocoding and location data using Google Maps API
  • jQuery transitions enhance user checkout experience
  • Twilio:order confirmations via text
  • Sendgrid:notifications via email
